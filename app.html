Attributes of Paragraph Tags

id: any string value,e.g Id="Intro Paragraph" class: any string value, e.g. class=“content-paragraph"
style: any valid CSS style value, e.g. style="color: blue; font-size: 32px;"
title: any string value, e.g. title=“Important Information"
dir: "ltr" (left-to-right) or "rtl" (right-to-left), e.g. dir="rtl"
lang: a valid language code, e.g. lang=“fr“
